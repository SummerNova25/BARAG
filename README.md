# BARAG
Hybrid Explainable Clinical Decision Support System combining ML-based diabetes prediction with Retrieval-Augmented Generation (RAG). Uses SHAP for model explainability and guideline-grounded LLM responses to generate structured clinical action plans (Diagnosis, Treatment, Follow-up, Referral) from official diabetes PDFs.
